Mars Weather Model¶

Pre-processing the Data¶

In order for the model to be accurate in its predictions, it will need to be fed clean and pre-processed data. This section of the notebook will focus on reading the data from the CSV file and cleaning the data so that only required data is fed into the model so that it can be trained to give an accurate output.

The summary table above shows us which columns need to be dropped in order for the data to be in a good format for the model to be processed with. For the purpose of this prediction model, the following columns will be dropped:

  1. id
  2. sol
  3. ls
  4. month
  5. min_temp
  6. pressure
  7. wind_speed
  8. atmo_opacity

This is because the prediction model will focus primarily on the terrestrial date and the max temperature of Mars at that date. The other columns will simply pollute the date and give us a less accurate prediction.

Now the null values have been removed but we can see from the figure above there is also another problem. The terrestrial date column currently holds the date in the form of an object, which cannot be processed by the model. In order to be fed through to the model with no issues, the data values in this column need to be converted to datetime objects so that the model can understand and process the data.

Training the Model¶

This section of the notebook will focus on training the model. For the purpose of this project, we will use the LSTM Prediction Model as this is a good model to use when you want to make predictions based on time-series datasets.

# Splitting the data into training and testing data, and then scaling the data using a min-max scaler.
data.index = data.terrestrial_date
data.drop("terrestrial_date", axis = 1, inplace = True)

final_data = data.values
train_data = final_data[0:1300, :]
valid_data = final_data[1300:, :]

scaler = MinMaxScaler(feature_range = (0, 1))

scaled_data = scaler.fit_transform(final_data)
x_train, y_train = [], []
for i in range(60, len(train_data)):
    x_train.append(scaled_data[i - 60:i, 0])
    y_train.append(scaled_data[i, 0])
    
x_train = np.asarray(x_train)
y_train = np.asarray(y_train)
In [38]:
# Using the LSTM prediction model.
lstm_model = Sequential()
lstm_model.add(LSTM(units = 50, return_sequences = True, input_shape = (np.shape(x_train)[1], 1)))
lstm_model.add(LSTM(units = 50))
lstm_model.add(Dense(1))

model_data = data[len(data) - len(valid_data) - 60:].values
model_data = model_data.reshape(-1, 1)
model_data = scaler.transform(model_data)

lstm_model.compile(loss = "mean_squared_error", optimizer = "adam", metrics=["MeanSquaredError", "AUC"])
lstm_model.fit(x_train, y_train, epochs = 10, batch_size = 5, verbose = 1)

x_test = []
for i in range(60, model_data.shape[0]):
    x_test.append(model_data[i - 60:i, 0])

x_test = np.array(x_test)
x_test = np.reshape(x_test, (x_test.shape[0], x_test.shape[1], 1))

predicted_max_temp = lstm_model.predict(x_test)
predicted_max_temp = scaler.inverse_transform(predicted_max_temp)
